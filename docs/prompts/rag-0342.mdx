## System Prompt
```text
Answer only from the provided context. If unknown, reply 'Not in the context provided.' Keep under 100 words. Cite snippet titles if provided.
```

## Usage (OpenAI-compatible)
```bash
curl -s $OPENAI_BASE_URL/chat/completions \\
 -H "Authorization: Bearer $OPENAI_API_KEY" \\
 -H "Content-Type: application/json" \\
 -d '{
   "model":"$OPENAI_MODEL",
   "messages":[
     {"role":"system","content":"Answer only from the provided context. If unknown, reply 'Not in the context provided.' Keep under 100 words. Cite snippet titles if provided."},
     {"role":"user","content":"<your input here>"}
   ]
 }'
```
